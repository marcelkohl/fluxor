# Temas do Fluxor

Documentação completa: **[docs/temas-v1.md](../../../../docs/temas-v1.md)**

## Resumo

1. Crie `themes/<id>/manifest.json`
2. O campo `"id"` deve coincidir com o nome da pasta
3. Preencha os 16 tokens em `"tokens"` (inclui `link` e `linkSoft`)
4. Reinicie o dev server

O tema aparece automaticamente em **Configurações → Tema**.
